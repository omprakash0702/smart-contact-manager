About Smart Contact Manager

Smart Contact Manager (SCM) is a cloud-based web application that helps users securely store, organize, and manage their personal and professional contacts from anywhere.

The application provides a simple and intuitive interface where users can add, update, delete, and search contacts efficiently. Each user has a personalized dashboard to manage their data securely.

SCM is built using modern technologies like Spring Boot, Spring Security, Thymeleaf, and MySQL, ensuring scalability, security, and performance.

Key Features

  • User authentication and secure login
  • Cloud-based contact storage
  • Search and filter contacts easily
  • User dashboard and profile management
  • Future support for Google and GitHub login

Why SCM?

Smart Contact Manager is designed to simplify contact management while keeping user data safe. It is ideal for students, professionals, and anyone who wants a reliable way to manage contacts online.
